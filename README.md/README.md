# Credit Scoring Model

## Project Overview

This project predicts an individual's creditworthiness using Machine Learning techniques. The model analyzes customer financial and demographic information to classify credit risk as Good or Bad.

## Dataset

German Credit Risk Dataset

Features:

* Age
* Sex
* Job
* Housing
* Saving Accounts
* Checking Account
* Credit Amount
* Duration
* Purpose

Target:

* Risk (Good / Bad)

## Machine Learning Models Used

1. Logistic Regression
2. Random Forest Classifier

## Results

### Logistic Regression

* Accuracy: 76.0%
* Precision: 77.5%
* Recall: 92.9%
* F1 Score: 84.5%

### Random Forest

* Accuracy: 76.5%
* Precision: 78.7%
* Recall: 91.5%
* F1 Score: 84.6%
